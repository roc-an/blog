# 三数之和

## （一）题目描述

给你一个包含 `n` 个整数的数组 `nums`，判断 `nums` 中是否存在三个元素 `a`，`b`，`c`，使得 `a + b + c = 0` ？请你找出所有和为 `0` 且不重复的三元组。

注意：答案中不可以包含重复的三元组。

**示例 1**：

```
输入：nums = [-1, 0, 1, 2, -1, -4]
输出：[[-1, -1, 2], [-1, 0, 1]]
```

**示例 2**：

```
输入：nums = []
输出：[]
```

**示例 3**：

```
输入：nums = [0]
输出：[]
```

**提示**：

* `0 <= nums.length <= 3000`
* ![-10^{5}](https://latex.codecogs.com/png.image?\dpi{110}%20-10^{5}) <= nums[i] <= ![10^{5}](https://latex.codecogs.com/png.image?\dpi{110}%2010^{5})

## 排序 + 双指针

题目要求“答案中不可以包含重复的三元组”，这里可以进行“模式识别”，**可以利用排序按照一定顺序去寻找所有解，这样就自然避免了重复答案**。

由此，可以先将数组排序，然后从左至右寻找答案。

如果先取到一个元素，那么剩下两个数的寻找就成了“两数之和”问题。

至此，问题转变成了：**在一个排序数组中，找到所有和为一个目标值的两数之和**。

双指针法：

在一个升序数组中，使用两个指针分别指向数组的首尾：

* 如果两数和 < 目标值，就让头指针后移，让和增大
* 如果两数和 > 目标值，就让尾指针前移，让和减小
